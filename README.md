# CLI Audio Player

Command line `node.js` audio player


## Used Software

- [aurora.js](https://github.com/audiocogs/aurora.js/)

## Supported Audio Format

- mp3
- ogg
- flac
- aac
- alac

## Getting Started

```
$ git clone https://github.com/yhbyun/audio-player.git
$ cd autio-player
$ npm install
$ node app.js audio-file
```
